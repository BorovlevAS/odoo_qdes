Dynamic Search Panel Toggle

An smart tool to control the display or hidden for search panel.

Starting from Odoo13, the system has expanded the search view and introduced a search panel function. The search panel is very useful for filtering and sorting and organizing business data, and it is a very valuable tool for improving business efficiency.

Therefore, in a large number of scenarios, we will expand through the search view and configure to enable this feature. However, sometimes in order to display more list fields, we hope to hide this area and display it quickly when needed. Moreover, different business objects require search panels of different widths, so we have developed a series of extensions for native functions. .

This module can quickly hide or show the search panel area by adding a toggle button.

The module for odoo15 version is ready!

    • Does the module support cross-version use ?

    No! Since the Odoo front-end has major changes in each version, all front-end modules do not support cross-version use.

    How do I perform a cross-version upgrade ?

    According to Odoo's official policy, starting from version 13, each new version of the module needs to be purchased separately, if you need to upgrade the Odoo version, you need to repurchase our corresponding version of the module .

    Is the module compatible with the enterprise version and the community version?

    Yes, the module works well with Odoo Enterprise as well as Community. .
